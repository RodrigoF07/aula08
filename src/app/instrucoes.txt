    Criar uma Web API Fake (A simulação de um banco de dados online)
Passo 01: Instalar o json-server: npm install -g json-server
    OBS: O comando acima devera ser executado dentro da pasta do projeto
Passo 02: Criar um arquivo com a extenção .json: ex: banco.json
Passo 03: Executar o comando: json-serve --watch
